# Detroit AI Consulting Co.

Marketing website for Detroit AI Consulting Co., built from the business plan for a Metro Detroit AI consulting business focused on small business strategy, implementation, and training.

## Files

- `index.html` - main one-page website
- `styles.css` - visual styling and responsive layout
- `script.js` - reveal-on-scroll interactions
