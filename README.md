# Sidecar Advisory

One-page marketing site for Sidecar Advisory, a boutique AI consulting firm focused on helping small businesses save time, cut costs, and grow with practical AI implementation.

## Stack

- `index.html` - page structure and content
- `styles.css` - brand styling, layout, and responsive behavior
- `script.js` - sticky navigation state on scroll
